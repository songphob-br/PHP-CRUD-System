# PHP CRUD System

A simple and clean **PHP CRUD (Create, Read, Update, Delete)** web application built with MySQL and Bootstrap.

---

## 📌 Project Overview

This project is a basic CRUD system developed using pure PHP and MySQL.  
It allows users to manage data through a clean and user-friendly interface.

This project was built for learning backend fundamentals and database integration.

---

## 🛠️ Tech Stack

- PHP (Native)
- MySQL
- HTML5
- CSS3
- Bootstrap
-  Laragon

---

## Features

- ➕ Create new records  
- 📄 Read / View data  
- ✏️ Update existing records  
- ❌ Delete records  
- 📱 Responsive UI (Bootstrap)

## ⚙️ Installation Guide

1. Clone the repository

```bash
git clone https://github.com/songphob-br/PHP-CRUD-System.git
```

2. Move project to your web server directory  
   (e.g. `htdocs` or `laragon/www`)

3. Create a MySQL database

4. Import the SQL file (if available)

5. Update database connection in:

```
config/db.php
```

6. Start Apache & MySQL

7. Open in browser:

```
http://localhost/PHP-CRUD-System
```
